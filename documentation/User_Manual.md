# System Admin & End-User Guide

## 1. System Administration Manual

### User Management
- **Location:** Admin -> Staff Management
- **Actions:**
    - Create new accounts for Cashiers.
    - View staff logs and performance.
    - Reset passwords by editing user records.

### Inventory Control
- **Location:** Admin -> Inventory / Product Manager
- **Adding Stock:** Use the **Purchases** section to link stock arrivals to Suppliers. This automatically updates inventory counts and financial balances.
- **Low Stock Alerts:** Monitor the dashboard and inventory list for items below their "Reorder Level".

### Financial Analytics
- **Location:** Admin -> Reports
- **Reports Available:**
    - Daily Sales Summary.
    - Supplier Payment Aging.
    - Profit/Loss Analysis.
    - Customer Account Statements.

## 2. End-User (Cashier) Manual

### Point of Sale (POS)
1. **Barcode Scanning:** Use the search bar or scan a barcode to add items to the cart.
2. **Customer Selection:** Choose a "Walk-in" customer or select an existing loyalty customer.
3. **Payments:**
    - **Cash/Card/UPI:** Instant checkout.
    - **Split Payments:** Allow customers to pay using multiple methods (e.g., half Cash, half UPI).
    - **Credit:** For trusted customers, select 'Credit' to add the amount to their balance.
4. **Receipts:** Upon completion, the system generates a professional invoice.

### Returns & Exchanges
- **Location:** Cashier -> Returns (or via Admin Dashboard)
- **Process:** Search by Invoice Number, select items to return, and process the refund (Cash or Credit).

## 3. Frequently Asked Questions (FAQ)

**Q: What if the stock count is wrong?**
A: Use the "Product Manager" to manually adjust stock or check "Purchases" history to see if an entry was missed.

**Q: How do I handle a customer wanting to pay their off-standing balance?**
A: Go to **Payments**, select the Customer, and record a "Payment In". You can then "Allocate" this payment to specific unpaid invoices.

**Q: The barcode scanner isn't adding items.**
A: Ensure the search input is focused and that the product's barcode is correctly saved in the "Product Manager".
